# CHANGELOG

## v2.5.0

### Added

- 進展管理ページの上部・左列・右列を分けた新しいGRIDレイアウトを追加しました。PCでは左列と右列が独立スクロールし、モバイル縦向きでは一列表示になります。
- マインドマップのルート表示設定を図書ごとに追加しました。「章から表示」と「図書名から表示」を選択できます。初期値は「章から表示」です。
- マインドマップをドラッグ・タッチ操作で移動できるよう、ECharts Treeにroam操作を追加しました。
- ナビゲーション右側のユーザー表示にUIkitドロップダウンを追加し、ユーザーセンター、図書管理、ログアウトをまとめました。
- ユーザー用の独立した図書管理ページを追加しました。ユーザーは自分で追加した図書だけを追加・修正・目次導入できます。
- 管理システムの図書管理を「図書追加」「目次導入」「登録図書」のuk-tab構成に変更しました。
- 図書追加・修正フォームに図書写真のプレビューを追加しました。
- 登録図書の修正はuk-modalで行えるようにし、保存・廃棄ボタンを追加しました。
- 目次進展リストに固定タイトル行を追加しました。

### Changed

- UIDログインを廃止し、ログイン方法をIDまたはメールのみに変更しました。
- ユーザーセンターでIDを変更できるようにしました。IDは16桁以内の半角英数字のみで、大文字・小文字だけが違うIDは同一扱いです。
- 「Timezone」を「タイムゾーン」に変更し、管理画面の権限・状態・チャート名などの表示を日本語化しました。
- 進展管理ページの「達成記録」を章節目次の下へ移動し、達成した項目を箇条書きで表示するようにしました。
- 進展パーセント欄、完成時間欄、メモボタンの幅と文字サイズを調整しました。
- 「図書の基本情報カード」を「図書情報」に変更しました。
- 「基本情報」を「進展状況」に変更し、左列の一番下に移動しました。
- 管理システムの「その他」メニューを削除しました。
- 管理システムのユーザー管理では、メールアドレスの `@` を `#` として表示するようにしました。

### Fixed

- CSPの `script-src` によるインラインスクリプトブロックが発生しないよう、インストーラーや確認処理のインラインJavaScriptを外部JSへ移動しました。
- HTMX自動保存後に章タブが第1章へ戻る問題を修正し、操作中の章タブを維持するようにしました。
- HTMX自動更新後に進展管理ページ左列のカード間に不要な空白が出やすい問題を、レイアウト再構成で抑制しました。

### Security

- ログイン方法からUIDを除外し、ID・メールのみで認証するようにしました。
- ユーザー用図書管理では `created_by` を検証し、自分が追加した図書以外を修正・目次導入できないようにしました。
- インラインイベント属性を削除し、CSPに適合しやすい外部JSイベントバインドへ整理しました。
- 図書画像アップロード、目次導入、メモ保存、HTMX自動保存、管理者操作の入力検証を再確認しました。

### SQL

- SQL変更あり。
- `books.mindmap_root` を追加しました。
- 既存DB用に `database-sample/migration_v2.5.0.sql` を追加しました。
- 既存環境は `php tool/upgrade/index.php` を実行してください。
